# Seam 6：输入工具栏升级与拾微附件上传面板

## Problem Statement

沉淀页富文本输入框工具栏功能不足，缺少通用上传、超链接、麦克风、标签、更多格式等快捷入口；拾微页附件上传入口仅为图片图标，无法表达已支持的图片/音频/视频/文件/链接等能力。

## Solution

升级沉淀页富文本输入框工具栏，并新增拾微页底部附件上传面板：
- 沉淀工具栏增加通用上传、超链接、麦克风、标签 `#`、更多（表格/代码块等）按钮。
- 拾微页点击附件按钮后从底部上滑半屏面板，提供相册、音频、视频、链接、文件、取消六个选项。

## User Stories

1. 作为用户，我希望沉淀页富文本输入框工具栏有通用上传按钮，支持图片/音频/视频，以便统一上传入口。
2. 作为用户，我希望工具栏有超链接按钮，点击后弹出输入框插入 Markdown 链接，以便快速加链接。
3. 作为用户，我希望工具栏有麦克风按钮，点击录音并转写后插入文本，以便语音输入。
4. 作为用户，我希望工具栏有 `#` 标签按钮，点击插入标签或弹出标签选择器，以便快速打标签。
5. 作为用户，我希望工具栏有“…”更多按钮，展开后包含表格、代码块、内联代码、导出、预览 Markdown，以便使用高级编辑功能。
6. 作为用户，我希望拾微页底部附件按钮点击后从底部上滑面板，面板内有相册、音频、视频、链接、文件、取消六个大图标选项，以便直观选择附件类型。
7. 作为用户，我希望点击面板外部或取消按钮都能关闭附件面板，以便灵活退出。

## Implementation Decisions

- **沉淀页富文本输入框工具栏**增加以下按钮（从左到右）：
  - 通用上传文件按钮：唤起文件选择器，accept `image/*,audio/*,video/*`，多选，走现有附件处理流程。
  - 超链接按钮：弹出输入框填写 URL 与显示文本，插入 Markdown 链接。
  - 麦克风按钮：点击开始录音，再次点击结束，走 STT 流程，转写文本插入光标处。
  - `#` 标签按钮：点击插入 `#` 或弹出标签选择器。
  - `…` 更多按钮：展开菜单包含表格、代码块、内联代码、导出、预览 Markdown。
- **拾微页底部附件上传面板**：
  - 点击附件按钮后，从底部上滑弹出半屏面板。
  - 面板选项：相册（图片）、音频、视频、链接、文件、取消。
  - 使用 lucide-react 大图标 + 文字标签。
  - 网格布局：第一行 3 个（相册/音频/视频），第二行 2 个功能项 + 取消按钮。
  - 取消按钮样式稍弱（灰色）。
  - 点击遮罩/外部或取消按钮关闭面板。

## Testing Decisions

- 验证沉淀工具栏各按钮触发对应行为：上传文件、插入链接、录音转写、插入标签、更多菜单。
- 验证拾微附件面板从底部上滑，六个选项布局正确，点击遮罩/取消关闭。
- 验证面板在移动端和桌面端均从底部弹出，不依赖 body 滚动。
- 验证文件选择器 accept 范围正确，多选后走现有附件处理流程。

## Out of Scope

- 不改动 STT/ASR 后端逻辑。
- 不改动附件存储与处理 pipeline。
- 多媒体卡片渲染规范见 Seam 4。
