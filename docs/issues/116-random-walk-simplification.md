# 随机漫步精简与视觉调整

## Parent

- #110 UI/UX 重构 v3 跟进（需求补漏）

## What to build

对随机漫步（`RandomWalk`）进行整体精简，减少视觉干扰并优化布局：

1. 单次抽取卡片数由 3 张改为 7 张（`DRAW_COUNT`）。
2. 移除顶部细栏中的 `1/N` 计数展示。
3. 移除顶部灯泡图标（`Lightbulb`）及其相关提示逻辑。
4. 将设置图标（`Settings2`）从当前位置移到原灯泡位置（左上角），点击仍打开数据源/冷却期配置面板，面板展开方向与位置随图标移动而调整。
5. 降低后方非当前卡片的透明度，避免前后文字重影影响阅读。
6. 卡片宽度缩小 1.2 倍，高度保持不变（仍 flex-1 填充可用空间）。
7. 移除底部操作栏的「已阅」按钮（`walk-read`）。
8. 剩余 5 个底部按钮（标签、编辑、复制、删除、换一批）调整 icon/文字大小与间距，使整体与缩小后的卡片宽度协调。
9. 「已阅」相关的 `handleRead`、`randomWalk.read` i18n、已阅状态过滤逻辑可保留在代码中供后续可能复用，但 UI 上不再展示。

该 slice 主要涉及 `RandomWalk.tsx` 的 UI 与样式调整，并同步更新 `tests/random-walk.test.ts`。

## Acceptance criteria

- [ ] 随机漫步一次展示 7 张卡片
- [ ] 顶部细栏无 `1/N` 计数展示
- [ ] 顶部无灯泡图标
- [ ] 设置图标位于左上角（原灯泡位置）且点击可打开配置面板
- [ ] 后方非当前卡片透明度降低
- [ ] 卡片宽度缩小 1.2 倍，高度不变
- [ ] 底部操作栏无「已阅」按钮
- [ ] 剩余 5 个底部按钮大小与间距与缩小后的卡片协调
- [ ] 滑动、换一批、编辑、复制、删除等核心功能保持可用
- [ ] `tests/random-walk.test.ts` 覆盖卡片数、计数/灯泡/已阅移除、设置图标位置、卡片尺寸与透明度
